# inline_logger Example

This example demonstrates all the features of the `inline_logger` package.

## Running the Example

1. Navigate to the example directory:
   ```bash
   cd example
   ```

2. Get dependencies:
   ```bash
   flutter pub get
   ```

3. Run the app:
   ```bash
   flutter run
   ```

## What's Demonstrated

- **Counter Example**: Shows inline logging in widget tree
- **API Call Simulation**: Demonstrates API request/response logging with timing
- **Log Levels**: All 7 log levels (Debug, Verbose, Info, Success, Warning, Error, Critical)
- **Inline Logging**: Chainable logging examples
- **Log History**: View stored logs

## Check Your Console

All logging output appears in your IDE console or terminal. Watch the console as you interact with the app to see the logging in action!
